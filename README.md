# SAP UI5 Quickstart

SAP UI5 Quickstart found [here](https://sapui5.hana.ondemand.com/#/topic/592f36fd077b45349a67dcb3efb46ab1).

## Run the application

1. Clone the repository

2. Install dependencies

```
npm install
```

3. Run the local server

```
ui5 serve
```

## More

```
npm install --global @ui5/cli

npm init --yes

ui5 init    
```
